Welcome to Neutron

Neutron provides a powerful set of tools for combining AutoHotkey with custom HTML user interfaces. Below are a few examples of ways you can leverage Neutron to your advantage.

Trigger AHK by page events

The simplest way of integrating a page to your script is by linking a button's click event to a function in your script. This can be easily achieved using the button's onclick attribute.

Click events aren't the only type that can be handled. The box below responds to mouse movement events and gets automatically updated with the cursor position by AutoHotkey.
